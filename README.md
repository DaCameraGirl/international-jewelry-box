💎 International Jewelry Box
Translation, but beautiful — neon aesthetics, floating gems, and 26 languages.
The International Jewelry Box is a growing collection of creative micro‑tools designed to make everyday digital tasks feel luxurious, playful, and visually alive.
The first compartment — the Translation Gem — transforms text into 26 languages with a glowing neon UI, floating gem animations, and a soft, immersive aesthetic.

More compartments (tools) will be added as the jewelry box expands.

✨ Features
26 Languages — smooth, fast translation with clean typography

Neon UI — glowing gradients, soft shadows, and gem‑like highlights

Floating Gem Animations — subtle motion that makes the interface feel alive

PWA‑Ready — installable on desktop & mobile

Lightweight & Fast — built with HTML, CSS, and a touch of JavaScript

Modular Design — each “compartment” is a standalone tool

🌍 Supported Languages
Arabic, Chinese (Simplified), Chinese (Traditional), Czech, Danish, Dutch, English, Filipino, Finnish, French, German, Greek, Hebrew, Hindi, Hungarian, Indonesian, Italian, Japanese, Korean, Norwegian, Polish, Portuguese, Romanian, Russian, Spanish, Swedish, Thai, Turkish, Ukrainian, Vietnamese
(26 total)
