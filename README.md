## 🌌 Overview

**International Jewelry Box** is a growing collection of creative, lightweight web tools designed to make everyday digital tasks feel elegant, luxurious, and visually alive. Instead of presenting utility tools in a plain or purely functional way, this project wraps them in a neon-inspired interface with glowing accents, floating gem aesthetics, sparkling motion, and a polished interactive feel.

The first tool in the collection, **Translation Gem**, transforms multilingual text translation into a more immersive experience through a jewel-box-inspired design system.

As the jewelry box expands, more “compartments” will be added — each one a standalone micro-tool with its own function, personality, and sparkle. ✨

---

## 💠 Current Compartment: Translation Gem

**Translation Gem** is the first active compartment inside International Jewelry Box. It allows users to translate text across multiple languages through a glowing neon interface enhanced by gem-like styling, floating visual elements, dancing diamonds, popping stars, and clean typography.

This compartment is designed to feel more magical, expressive, and polished than a standard translation tool while remaining lightweight and easy to use.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/DaCameraGirl/international-jewelry-box.git
```

### 2. Open the Project

Open `index.html` directly in your browser.

No build step is required. ✨

### 3. Optional Local Server

If you would like to serve the project locally:

```bash
npx serve .
```
